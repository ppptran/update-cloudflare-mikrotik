# update-cloudflare-mikrotik
Script running in Mikrotik Router to update Public IP to cloudflare DNS

 There are 2 scrips. Ipv4 & Ipv6.

 For Ipv6, there're some Logs and comment in Vietnamese, just copy it and ask Geminy to translate guys. I'm too lazy.
 Note for Ipv6. The Public Ip interface is detected in my setup is BridgeHome, change it to whatever you guys use.

 Using Mikrotik Terminal, print the Ipv6 -> Address , copy & paste to AI, it will show you which one to select.
